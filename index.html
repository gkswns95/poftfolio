HC

I received a B.S. and an M.S. in Computer and Communication Engineering from Kangwon National University, advised by Prof. Sang-Ki Ko.

Research Objectives

Implementation of a deep learning framework to solve missing trajectory problems in spatiotemporal data acquired from multi-agent environments such as soccer, basketball, and American football

Core Contributions

  1. Proposed a position interpolation neural network considering permutation equivariance of input data (1st prediction)
  2. Proposed a position prediction algorithm based on velocity and acceleration accumulation (2nd prediction)
  3. Proposed a dynamic fusion neural network for 1st and 2nd position predictions according to missing length and situations

Responsibilities

  • Preprocessing spatiotemporal data and building algorithms for soccer, basketball, and American football
  • Designing and implementing velocity and acceleration-based position prediction models
  • Designing and implementing dynamic fusion models for position predictions
  • Implementing baseline models and conducting comprehensive experiments and result analysis
  • Paper writing and conference presentation

KIBME 2025 (Best Paper)

Multimodal Sentiment Analysis under Incomplete Modalities with Missing Text Localization

Han-Jun Choi, Byunggill Joe, Jin Yea Jang

We address the challenge of multimodal sentiment analysis when text modalities are incomplete or missing. Our confidence-aware fusion strategy effectively localizes missing text segments and compensates for them using other modalities.

  • Introduced a confidence-aware multimodal fusion layer.
  • Developed a missing text localization algorithm.
  • Won Best Paper Award at KIBME 2025.
    • Multimodal Sentiment Analysis Deep Learning Fusion

    KDD 2023

    Ball trajectory inference from Multi-Agent Sports Contexts Using Set Transformer

    Hyunsung Kim, Han-Jun Choi, Chang-Jo Kim, Jinsung Yoon, Sang-Ki Ko

    Inferring the ball trajectory in sports videos is challenging due to occlusions and small object size. We propose a Set Transformer and Hierarchical Bi-LSTM based approach to infer ball positions from player contexts.

  • Utilized Set Transformer to model player interactions.
  • Hierarchical Bi-LSTM for temporal consistency.
  • Published at KDD 2023 (Top Tier).
    • Sports Analytics Transformer Bi-LSTM Computer Vision

    CIKM 2025

    Imputing Multi-Agent Trajectories from Event and Snapshot Data in Soccer

    Geonhee Jo, Miru Hong, Han-Jun Choi, Pascal Bauer, Sang-Ki Ko

    We combine event data and snapshot data to reconstruct complete multi-agent trajectories in soccer matches. This hybrid approach overcomes the limitations of using either data source alone.

  • Hybrid data fusion of events and snapshots.
  • Improved trajectory reconstruction accuracy.
  • Applicable to various team sports.
